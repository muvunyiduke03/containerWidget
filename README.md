# Flutter Container Widget
This is a simple mobile application that demonstrates the container widget. This is a versatile widget that combines sizing, spacing, alignment and decoration by a single widget.

## How to run the application
1. Install flutter in your IDE.
2. Clone the repository:
   ```
   git clone https://github.com/muvunyiduke03/containerWidget.git
   cd Your_Repo_Name
   ```
3. Run the app:
  ```
  flutter run
  ```

## Key Attributes

1. Width and height
   These attributes controls the size of the container widget. Height controls the vertical size.    Width controls the  horizontal size of the container widget.
   
2. ### Margin and padding
   - Margin: Adds space on the outside of the container.
   - padding: Adds space on the inside of the widget between the borders and its child.

3. ### decoration
   Applies styling to the container such as: color, borderRadius, boxShadow, etc...
